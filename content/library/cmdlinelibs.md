# Command-line interface libraries

The modules described in this chapter assist with implementing
command line and terminal interfaces for applications.

Here’s an overview:

* [`argparse` — Parser for command-line options, arguments and subcommands](argparse.md)
* [`optparse` — Parser for command line options](optparse.md)
* [`getpass` — Portable password input](getpass.md)
* [`fileinput` — Iterate over lines from multiple input streams](fileinput.md)
* [`curses` — Terminal handling for character-cell displays](curses.md)
* [`curses.textpad` — Text input widget for curses programs](curses.md#module-curses.textpad)
* [`curses.ascii` — Utilities for ASCII characters](curses.ascii.md)
* [`curses.panel` — A panel stack extension for curses](curses.panel.md)
* [`cmd` — Support for line-oriented command interpreters](cmd.md)
