# `urllib` — URL handling modules

**Source code:** [Lib/urllib/](https://github.com/python/cpython/tree/main/Lib/urllib/)

---

`urllib` is a package that collects several modules for working with URLs:

* [`urllib.request`](urllib.request.md#module-urllib.request) for opening and reading URLs
* [`urllib.error`](urllib.error.md#module-urllib.error) containing the exceptions raised by [`urllib.request`](urllib.request.md#module-urllib.request)
* [`urllib.parse`](urllib.parse.md#module-urllib.parse) for parsing URLs
* [`urllib.robotparser`](urllib.robotparser.md#module-urllib.robotparser) for parsing `robots.txt` files
