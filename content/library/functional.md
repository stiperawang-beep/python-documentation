# Functional Programming Modules

The modules described in this chapter provide functions and classes that support
a functional programming style, and general operations on callables.

The following modules are documented in this chapter:

* [`itertools` — Functions creating iterators for efficient looping](itertools.md)
  * [Itertool Functions](itertools.md#itertool-functions)
  * [Itertools Recipes](itertools.md#itertools-recipes)
* [`functools` — Higher-order functions and operations on callable objects](functools.md)
  * [`partial` Objects](functools.md#partial-objects)
* [`operator` — Standard operators as functions](operator.md)
  * [Mapping Operators to Functions](operator.md#mapping-operators-to-functions)
  * [In-place Operators](operator.md#in-place-operators)
