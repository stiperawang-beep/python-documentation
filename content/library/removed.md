# Removed Modules

The modules described in this chapter have been removed from the Python
standard library.  They are documented here to help people find replacements.

* [`aifc` — Read and write AIFF and AIFC files](aifc.md)
* [`asynchat` — Asynchronous socket command/response handler](asynchat.md)
* [`asyncore` — Asynchronous socket handler](asyncore.md)
* [`audioop` — Manipulate raw audio data](audioop.md)
* [`cgi` — Common Gateway Interface support](cgi.md)
* [`cgitb` — Traceback manager for CGI scripts](cgitb.md)
* [`chunk` — Read IFF chunked data](chunk.md)
* [`crypt` — Function to check Unix passwords](crypt.md)
* [`distutils` — Building and installing Python modules](distutils.md)
* [`imghdr` — Determine the type of an image](imghdr.md)
* [`imp` — Access the import internals](imp.md)
* [`mailcap` — Mailcap file handling](mailcap.md)
* [`msilib` — Read and write Microsoft Installer files](msilib.md)
* [`nis` — Interface to Sun’s NIS (Yellow Pages)](nis.md)
* [`nntplib` — NNTP protocol client](nntplib.md)
* [`ossaudiodev` — Access to OSS-compatible audio devices](ossaudiodev.md)
* [`pipes` — Interface to shell pipelines](pipes.md)
* [`smtpd` — SMTP Server](smtpd.md)
* [`sndhdr` — Determine type of sound file](sndhdr.md)
* [`spwd` — The shadow password database](spwd.md)
* [`sunau` — Read and write Sun AU files](sunau.md)
* [`telnetlib` — Telnet client](telnetlib.md)
* [`uu` — Encode and decode uuencode files](uu.md)
* [`xdrlib` — Encode and decode XDR data](xdrlib.md)
