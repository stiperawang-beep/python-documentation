# Development Tools

The modules described in this chapter help you write software.  For example, the
[`pydoc`](pydoc.md#module-pydoc) module takes a module and generates documentation based on the
module’s contents.  The [`doctest`](doctest.md#module-doctest) and [`unittest`](unittest.md#module-unittest) modules contains
frameworks for writing unit tests that automatically exercise code and verify
that the expected output is produced.
